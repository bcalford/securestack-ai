# Roadmap

## Implemented

- Local Docker Compose runtime.
- React/TypeScript/Vite frontend and Java 21/Spring Boot backend.
- Defensive static analysis rules.
- Results dashboard, finding filters/status, scan history, PDF export, and SARIF export.
- Mock AI by default.
- Optional manually configured Amazon Bedrock summaries.
- Guided sample review flow.
- AWS architecture blueprint.
- Frontend unit tests with Vitest and Testing Library.
- Optional local PostgreSQL Docker profile for persistence validation.

## Future

- Authentication and authorization.
- GitHub repository scanning.
- OpenAI provider.
- Semgrep integration and SARIF ingestion/import.
- Production AWS deployment automation.
- Persistent multi-user storage.
- Hosted demo with abuse controls.
