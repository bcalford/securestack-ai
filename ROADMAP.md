# Roadmap

- Pasted/uploaded file scans.
- Static heuristic rules.
- Mock AI summaries.
- Results dashboard and scan history.
- Finding filters and status updates.
- PDF report export.
- Docker Compose local runtime.

- Authentication and user-owned scans.
