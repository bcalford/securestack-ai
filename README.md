# SecureStack AI

SecureStack AI is an AI-assisted security review platform for small software projects.

It scans uploaded or pasted project files for common security risks across application code, dependencies, API configuration, Dockerfiles, cloud/IaC files, and secrets handling. It produces a dashboard, prioritized findings, AI-assisted summaries, and exportable PDF reports.

## Planned stack

- React
- TypeScript
- Java 21
- Spring Boot
- Docker
- GitHub Actions
- Mock AI provider by default
- Optional OpenAI-compatible or Amazon Bedrock provider
